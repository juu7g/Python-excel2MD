# Python-excel2MD

## 概要 Description
エクセルファイルを読んでMarkdown記法の表に変換してテキストファイルに出力します。  
Read the Excel file, convert it to a Markdown notation table, and output it to a text file.   

## 特徴 Features

- エクセルの最初のシートを対象にします。  
	Targets the first sheet of Excel   
- 1行目は見出しとします  
	The first line is the heading.  

## 依存関係 Requirement

- Python 3.8.5  
- pandas 1.2.2  
- openpyxl 3.0.6
